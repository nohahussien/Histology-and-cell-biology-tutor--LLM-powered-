# Histology & Cell Biology Tutor (LLM-powered)

A Flask-based web application that allows users to ask questions related to
Histology and Cell Biology using a Large Language Model (Groq API).
All questions and answers are stored in a PostgreSQL database.

## Features
- Domain-restricted LLM (Histology & Cell Biology)
- Persistent QA history
- PostgreSQL database
- Dockerized application
- Deployed-ready (Render compatible)

## Requirements
- Docker
- Docker Compose

## Run with Docker

```bash
docker compose up --build
